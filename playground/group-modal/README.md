# Group Modal (Prototype)

Prototype modal for **Group Admin → Group Info** that supports adding/editing a group.

## Figma

- **GroupInfoPage context**: `https://www.figma.com/design/W0qzhXWxFsMwSJzkdV2yal/Design-System---Web-App-Specs?node-id=258-263800`
- **Modal shell reference (TutorModal)**: `https://www.figma.com/design/W0qzhXWxFsMwSJzkdV2yal/Design-System---Web-App-Specs?node-id=258-262330`

## What’s included

- Group name (`Input`)
- School (`Select`, single)
- Tutors (`Select`, multi w/ badges)
- Size (`NumberInput`)

## Assumptions (prototype)

- This is a **visual + interaction demo** (not production wiring): “Save/Delete” log to console.
- School + tutor lists are mocked in Storybook.

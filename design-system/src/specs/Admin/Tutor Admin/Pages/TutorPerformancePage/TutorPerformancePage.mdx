### Anatomy

The page is built on the PLUS `PageLayout` shell. Top-down:

- **TopBar** with breadcrumbs (`Admin / Tutor Admin / Tutor Performance`).
- **Sidebar** for primary navigation.
- **Tab navigation** — four tabs: `Tutor Performance`, `Status And Warnings`, `Tool Usage`, `Training Progress`.
- **Action buttons** — `Email Tutors` and `Export Reflection Data` (top-right of the workspace).
- **Performance Overview section** — filter bar plus two donut charts: Attendance and Sign-Up Rate.
- **Performance Details table** — sortable, paginated, with row-click → tutor modal.

### Modal state

The page exposes `modalOpen`, `modalTab` (`info` / `sessions`), and `modalMode` (`edit` / `add`) so the embedded tutor modal can be opened directly. Because the modal renders over the entire viewport, the dedicated story is not embedded inline here — use the **Interactive playground** below and toggle `modalOpen` to see it in context.

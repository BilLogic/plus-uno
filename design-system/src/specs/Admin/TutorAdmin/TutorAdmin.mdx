The Tutor Admin spec is the unified surface for monitoring tutor performance, status, tool usage, and training progress. It bundles pages, sections, cards, tables, modals, and elements into one place — every block below renders inline with its own controls so you can poke at props without leaving the page.

### Modals

Modal stories render in an overlay layer, so they're listed here but linked out to their dedicated sidebar entries instead of embedded inline (otherwise every modal would auto-open over the doc body).

- **Tutor modal** — two-tab modal (Info / Sessions) with edit and add modes. → `Specs/Admin/Tutor Admin/Modals/TutorModal` (variants: `EditTutor`, `EditTutorSessions`, `AddNewTutor`).
- **Delete tutor modal** — destructive confirmation dialog for removing a tutor record. → `Specs/Admin/Tutor Admin/Modals/DeleteTutorModal` (variant: `Default`).

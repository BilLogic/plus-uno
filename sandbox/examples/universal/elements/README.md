# Universal Elements - Examples

## Overview
Examples of universal element components including buttons, checkboxes, alerts, and SMART components.

## Components

### Button
- [Basic Button Examples](./button.html)
- Variations: Filled, Outline, Tonal, Text
- Styles: Primary, Secondary, Tertiary, Success, Info, Warning, Error
- Sizes: Small, Default, Large
- States: Default, Hover, Active, Disabled

### Checkbox
- [Checkbox Examples](./checkbox.html)
- Single checkbox
- Checkbox groups
- States: Checked, Unchecked, Disabled

### Alert
- [Alert Examples](./alert.html)
- Styles: Primary, Secondary, Success, Danger, Warning, Info
- With/without dismiss button
- With/without title

### Status Icon
- [Status Icon Examples](./status-icon.html)
- Statuses: Assigned, Started, Not Started, Complete
- With SMART framework colors

### Content Status Tag
- [Content Status Tag Examples](./content-status-tag.html)
- Status indicators with text
- SMART framework integration

### Super Comp Pill
- [Super Comp Pill Examples](./super-comp-pill.html)
- Competency area pills
- Full and abbreviated versions
- SMART framework colors

## Token Usage

Each example highlights the design tokens used:
- **Colors**: Material Design 3 color tokens
- **Spacing**: Semantic spacing tokens (element-*)
- **Typography**: Typography tokens
- **Border Radius**: Element radius tokens
- **Border/Stroke**: Element stroke tokens

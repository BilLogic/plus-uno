# Foundation: Typography

**Context**: Global typography tokens.
**Layer**: Foundation

## 1. Font Families
| Token | Value | Usage |
| :--- | :--- | :--- |
| `var(--font-family-header)` | Lato | Headings, Titles |
| `var(--font-family-body)` | Merriweather Sans, Open Sans | Body text |
| `var(--font-family-code)` | Source Code Pro | Code blocks |

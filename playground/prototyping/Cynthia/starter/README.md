# Cynthia Starter Prototype

This is your new React/Vite playground project under `playground/prototyping/Cynthia/starter`.

## Run locally

```bash
cd playground/prototyping/Cynthia/starter
npm install
npm run dev
```

## What is included

- React + Vite starter setup
- PLUS design system alias (`@` -> `packages/plus-ds/src`)
- Token-based starter styles in `src/App.scss`
- Starter page in `src/App.jsx`

## Next steps

1. Edit `src/App.jsx` to build your first screen.
2. Keep styling token-based (`var(--color-*)`, `var(--size-*)`).
3. Reuse PLUS components from `@/components`.

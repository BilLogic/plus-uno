# Foundation: Colors

**Context**: Global color tokens (Material Design 3).
**Layer**: Foundation

## 1. Brand Colors
| Role | Token | Hex | Usage |
| :--- | :--- | :--- | :--- |
| **Primary** | `var(--color-primary)` | #0472a8 | Main brand color |
| On Primary | `var(--color-on-primary)` | #ffffff | Text on primary |
| Container | `var(--color-primary-container)` | #61b5cf | Lighter container |
| **Secondary** | `var(--color-secondary)` | #445c6a | Secondary elements |
| **Tertiary** | `var(--color-tertiary)` | #0e8175 | Accents |

# PLUS Design System Documentation

Welcome to the developer documentation for the PLUS Design System. This directory is organized to help you find exactly what you need without context overload.

## 🚀 Getting Started

### Which workflow matches your goal?

| I want to... | Go to... |
| :--- | :--- |
| **Develop** React Components/Pages | Run `npm run dev:react` |
| **View** Component Library | Run `npm run storybook` |
| **Migrate** Legacy Specs | Reference `src/components/` |

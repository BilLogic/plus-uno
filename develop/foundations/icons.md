# Icons

PLUS Design System uses **Font Awesome 6 Free** (Solid) for iconography.

## Installation (CDN)

Add the following link to your HTML `<head>`:

```html
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.7.2/css/all.min.css" integrity="sha512-Evv84Mr4kqVGRNSgIGL/F/aIDqQb7xQ2vcrdIwxfjThSH8CSR7PBEakCr51Ck+w+/U6swU2Im1vVX0SVk9ABhg==" crossorigin="anonymous" referrerpolicy="no-referrer" />
```

## Usage

Use the `i` tag with `fa-solid` and the icon name.

```jsx
<i className="fa-solid fa-user"></i>
```

Refer to [Font Awesome Search](https://fontawesome.com/search?o=r&m=free&s=solid) for available icons.

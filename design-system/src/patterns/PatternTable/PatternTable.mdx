<Title />
The table pattern is a shell for data presentation: a toolbar slot, the table content slot (you pass your own `thead`/`tbody`), a pagination slot, and an empty-state slot. The table itself is no-fill — the surrounding card or surface owns the background — and row interaction uses additive `on-surface` overlays, never replaced fills. It mirrors the Table slot component in Figma.

### Anatomy

- **Header** with the student's name and a close button.
- **Tab navigation** — `Info` and `Sessions`.
- **Info tab body** — profile fields (name, school, cohort, contact).
- **Sessions tab body** — session log table.
- **Footer actions** — `Cancel` and a primary `Save` button.

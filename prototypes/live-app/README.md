# Live app (Storybook specs replica)

Holistic product shell on production at `/home` (alias `/app`).

Assembles documented **Specs** page components from `design-system/src/specs/` into one navigable flow — not the recording-oriented `home-redesign` demo.

```bash
npm run dev:live-app
# or via root SPA
npm run dev:vite   # then open /home
```

Section switcher (top-right) jumps between Storybook page specs. Expand routes as new Specs pages land.

# Assets Folder Policy

This folder stores reusable, non-guidance artifacts for agents.

Allowed here:
- Structured lookup/index artifacts (for example `*.json`) for components, tokens, examples, and integrations.
- Reusable templates/snippets that may be copied into generated output.

Not allowed here:
- Narrative reference guidance markdown (except this policy file).

Guidance and workflow docs belong in `.agent/references/`.
